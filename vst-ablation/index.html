结论速览

1. 🗺 场景维度：HyperSim（峰值 37.24）与 Taskonomy（终点 36.95）单场景即超 native MI，ScanNet++/Waymo 也强劲；Replica/MP3D（Habitat 系）最弱（<34）——场景选择比混合更关键，合成/多任务场景迁移性最好；
2. 🏆 题型维度 · B2 视频几何双杀 native MI（峰值 36.90 / 终点 36.52 vs 36.76 / 36.14）——唯一终点超过混合臂的题型臂；
3. 📷 相机五题型互补：最强 cam_rot 终点 35.76 仍低于混合，混合收益 = 相机主体 ~+0.7 + 题型互补 ~+0.3；
4. 🧊 纯对象题 ≈ 没加（终点 34.95 vs baseline 35.10），但 对象×相机耦合立刻有效（obj_cam 峰值 35.76）；
5. ⏱ 对象臂后段才发力（obj_cam s600 33.90 → s900 35.76），早停会误判；多数臂 s900 > s1000，下游建议按验证集选点。
