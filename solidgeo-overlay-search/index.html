Solid Geometry Overlay Search

A rule-based augmentation route for solid geometry diagrams. The base geometry stays fixed; auxiliary points, lines, and faces are generated from a construction grammar, scored for readability and problem value, then selected as diverse overlay variants.

These examples target projection distance, section cuts, dihedral witnesses, and skew-line relations while keeping the diagram compact enough for visual QA data generation.

Line Relation

Skew With Side Point

Places a side-edge point and relation lines that expose non-coplanar line reasoning without cluttering the base shape.

1. Generate

Enumerate legal auxiliary constructions from a small geometry grammar instead of hand-picking one overlay at a time.

2. Score

Rank candidates by regularity, readability, exam-likeness, and visual simplicity.

3. Select

Choose top variants across different construction families to improve coverage without repeating the same visual pattern.
